Comprehensive Introduction to Computer Science

Module 1: Introduction to Computers

A computer is an advanced electronic device that takes in raw data as input, processes it using a set of instructions (programs), stores the results, and outputs meaningful information.

The Four Primary Functions (IPSO)

  1. Input: Gathering data (e.g., typing on a keyboard).
  2. Processing: Manipulating data into information (handled by the CPU).
  3. Storage: Saving data and information for later use.
  4. Output: Displaying the processed information (e.g., on a monitor).

The Five Generations of Computers

Computers have evolved dramatically over the decades, categorized into "generations" based on the core technology powering them:

  • First Generation (1940s–1950s): Powered by Vacuum Tubes. They were massive (taking up entire rooms), expensive, generated massive heat, and used punch cards for input. (Example: ENIAC)
  • Second Generation (1950s–1960s): Powered by Transistors. This made computers smaller, faster, cheaper, and more energy-efficient than their predecessors.
  • Third Generation (1960s–1970s): Powered by Integrated Circuits (ICs). Transistors were miniaturized and placed on silicon chips, drastically increasing speed and efficiency.
  • Fourth Generation (1971–Present): Powered by Microprocessors. Thousands of integrated circuits were built onto a single silicon chip. This era birthed the PC.
  • Fifth Generation (Present & Future): Powered by Artificial Intelligence (AI) and Quantum Computing.

Module 2: Computer Hardware

Hardware refers to the physical, tangible parts of the computer that you can touch and see.

1. The Central Processing Unit (CPU)

Often called the "brain" of the computer, the CPU executes instructions. It has three main parts:

  • ALU (Arithmetic Logic Unit): Performs all mathematical calculations (addition, subtraction) and logical comparisons (AND, OR, NOT).
  • CU (Control Unit): Directs the flow of data. It fetches, decodes, and executes instructions from memory.
  • Registers: Tiny, ultra-fast storage locations inside the CPU used during processing.

2. Computer Memory

Memory is where data is kept while the computer is running.

  • Primary Storage (Main Memory):
    • RAM (Random Access Memory): Volatile memory. Holds data temporarily.
    • ROM (Read-Only Memory): Non-volatile memory. Holds crucial boot-up system instructions.
  • Secondary Storage: Long-term, non-volatile storage. (HDDs, SSDs, USB flash drives).

3. Input & Output (I/O) Devices

Device Type Function Examples
Input Sends data into the computer. Keyboard, Mouse, Scanner
Output Sends information out of the computer. Monitor, Printer, Speaker

Module 3: Computer Software

Software is the set of intangible instructions that tells the hardware exactly what to do. You cannot physically touch software.

System Software

The foundational software that manages the computer's hardware and provides a platform for other software to run.

  • Operating Systems (OS): Windows, macOS, Linux, Android.
  • Utility Programs: Antivirus software, file backup tools.
  • Device Drivers: Small programs that allow the OS to communicate with specific hardware.

Application Software

Programs designed to help the end-user perform specific tasks.

  • Productivity: Microsoft Word, Excel, PowerPoint.
  • Browsers: Google Chrome, Mozilla Firefox.
  • Entertainment: Video games, Spotify, Netflix.

Module 4: Data Representation

How does a computer actually read all this software? Through Binary.

Computers do not understand human languages; they only understand electricity being "ON" or "OFF". We represent this using the Binary Number System (Base 2), which consists of only two digits: 0 (Off) and 1 (On).

  • Bit (Binary Digit): The smallest unit of data in computing (a single 0 or 1).
  • Byte: A grouping of 8 bits. One byte is enough to represent a single character, like the letter 'A' or the number '7'.

Data Size Hierarchy:

1 KB 1,024 Bytes
1 MB 1,024 KB
1 GB 1,024 MB
1 TB 1,024 GB

Module 5: Introduction to Computer Networks

A network is two or more computers connected together to share resources (like files, printers, or an internet connection).

LAN (Local Area Network)

Covers a small geographic area, like a single home, office, or school building.

WAN (Wide Area Network)

Covers a large geographic area, connecting multiple LANs across cities or countries. The Internet is the largest WAN in the world.
